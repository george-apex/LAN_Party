# Videos Folder

Place your MP4 video files in this folder.

**Required file:**
- `video.mp4` - This is the video that will play on the video screen in the Games room.

**How to use:**
1. Download or create an MP4 video file
2. Rename it to `video.mp4`
3. Place it in this folder
4. Go to the Games room in the game
5. Walk to the video chair (near the video screen on the left side)
6. Press F to sit down
7. Press SPACE to play/pause the video
8. Press F again to stand up

**Video recommendations:**
- Resolution: 640x360 or similar (16:9 aspect ratio)
- Duration: Any length (video will loop)
- File size: Under 50MB recommended for faster loading
